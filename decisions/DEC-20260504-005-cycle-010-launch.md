## E2: cycle 010 ordering -> parallel (Memory + Permanence)

Three reasons:

1. **Cycle 009 cross-pair pattern verified**: CRITIC-02 <-> COMPOSER-01 closed CR-1 via in-cycle forward-reference with no friction; the same pattern will close CR-2 via PERMANENCE-01 <-> MEMORY-01 (Permanence publishes `suppress_static_write`, Memory consumes per CR-2 binding rule). Both cards drafted in cycle 010 means CR-2 is exercised in-cycle without cross-cycle forward references.
2. **Memory-first is structurally worse for CR-2**: CR-2 is unidirectional Permanence -> Memory binding. If Memory drafted first, it would forward-reference an unwritten Permanence signal — a worse shape than the cycle 009 forward-reference null protocol from CRITIC-03 (which had a documented fallback path). Permanence-first is symmetric to Memory-first but better than Memory-first; parallel dominates both because cross-pair lets each card forward-reference the other when needed.
3. **Tempo**: user previously stated "全力提速了" (cycle 008.5 user direction). Parallel = one cycle wallclock for two finalists; sequential = two cycles for the same coverage. Parallel matches the tempo posture.

Forward-reference null protocol from `cases/CASE-20260504-CRITIC-03.md` is the canonical template for any CR-X read where the producer card has not yet landed within the cycle: read returns null + fallback path is documented in the reading card. Cycle 010 closeout audit must close every forward-reference null with the producer card's actual published value.

## E3: D3 first teacher demo target -> agent recommends continued deferral

Both deferral conditions from `decisions/DEC-20260504-002-no-all-in-on-single-finalist.md` are now satisfied (audience profile populated; cycle 009 case-card data exists). The user delegated this decision to the agent ("3 我觉得你决定就好"). Agent decision: **continue deferring D3 until cycle 010 closeout**, when all 4 finalists will have L2 case-card coverage.

Rationale:

- Currently 2 of 4 finalists (Critic + Composer) have L2 cards; 2 of 4 (Memory + Permanence) have none.
- Picking a demo target now would be picking under partial portfolio coverage. Per Discipline rule 5 (Honesty Override), "no L2 evidence" must not be silently equated with "weak option"; selecting on partial coverage builds in a portfolio-asymmetry bias that paper-derived case cards alone cannot correct.
- The teacher audience profile populated by the user signals "cold start, first impression, no specific 3R praise or criticism". Under cold-start framing, the demo target choice should optimize for the strongest first-impression material across all four finalists; that material requires symmetric L2 coverage.
- Cost of waiting one more cycle: the 90-120-minute cycle 010 budget. Cost of picking now: irreversibly anchoring the demo on the half of the portfolio that happened to draft first.

If the user wishes to override this agent decision and pick D3 now (e.g. for external-deadline reasons), this DEC accepts that override; the user can record the override in `decisions/DEC-20260504-006-d3-demo-target-pick.md` (id reserved here) at any time.
