## 一、开题报告主旨

本开题报告围绕一种面向前馈式三维重建（3R）的候选多模块架构（以下称"候选架构 X"），系统阐述其研究背景、研究问题、技术方案、实验设计与风险分析。候选架构 X 在现有 DUSt3R / MASt3R 基础上，引入六模块设计（感知 / 记忆 / 永久性 / 校验 / 编排 / 总线），尝试在长序列内存、几何校验与多专家组合三个方向上提供候选解决路径。

## 三、与同期中文综述的关系

本开题报告与同期完成的前馈式三维重建中文综述（18 页 / 44 引 / 10 章）为平行制品。两者共享引文池与部分研究现状素材，但：

1. **综述**聚焦该方向 2024 年以来的系统分类（六类失败模式 / 四类长序列内存 / 三类测试时机制 / 三类输出资产），不涉及候选架构 X 本身
2. **开题报告**以候选架构 X 为中心，将综述的分类框架作为研究定位的上下文，并在此基础上提出三组研究问题与六模块技术方案

两份文档独立成稿，不互相引用。

## 四、候选性与证据边界声明

本开题报告严格采用"候选不等于最终"的学术表述：

- 候选架构 X 是"被评估的候选方案"，不是已被验证的最终设计
- 所有技术声明附带证据标签（paper-derived / engineering-demonstrated / code-observed）；当前阶段无 paper-proven 声明
- 已完成的实现里程碑（1-18）产出的集成证据仅证明"张量契约可跑通"，不等于"训练后指标达标"
- 时间安排（M1-M8）为候选时间表，非 committed schedule；受算力、导师反馈与实证结果影响

## 五、已识别风险摘要

开题报告第 9 章梳理了架构层、跨模块、实证执行、工程时序四类风险，每条风险有方案级或部分缓解路径，但无一在开题阶段被完全消除。这一状态与候选研究定位一致。
