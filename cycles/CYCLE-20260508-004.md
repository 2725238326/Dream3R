## Main result

Cycle 027 converts the cycle 026 recommendation into a concrete P0 planning
artifact:

```text
planning/MEMORY_V03_P0_PROTOTYPE_PLAN.md
```

The plan compares four variants:

```text
V0 vector AnchorBank baseline
V1 Spann3R-style spatial key/value bank
V2 CUT3R-style state-token recurrence
V3 hybrid state tokens + spatial bank + bus-gated writes
```

The plan defines:

- deterministic tensor dimensions and seeds
- synthetic regimes R1 stable loop, R2 monotonic drift, R3 dynamic pollution, R4 conflict quarantine, R5 pressure and pruning
- fixture tensors and bus fields
- per-variant interfaces and expected weaknesses
- metrics M1-M8
- kill conditions K1-K7
- reviewer checklist
- future execution gate

server model path by default. A later implementation should first live under a
non-model prototype path, for example:

