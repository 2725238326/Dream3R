## 一、综述主旨

本综述系统梳理前馈式三维重建（feed-forward 3D reconstruction，简称 3R）研究方向自 2024 年 DUSt3R 发表以来的演化路径。3R 区别于经典 SfM/MVS 在于：以单次神经网络前向传播直接回归像素级三维点图与相机参数，跳过特征匹配、姿态估计、三角化等级联阶段。综述在以下四个维度上对该方向作系统分类：六类典型几何失败模式、长序列内存的四类机制、测试时一致性 / 适应 / 先验注入的三类区分、输出资产的三类（点图 / 动态掩码 / 高斯渲染）区分。

- 引文：44 篇（含 arXiv 2024 / 2025 / 2026 预印本与正式出版条目）

- 时间窗口：2024-01 ~ 2026-05
- 章节：10 章（引言 / 表示转变 / DUSt3R-MASt3R 基础 / 多视角规模化 / 视频与动态 4D / 长序列内存 / 测试时机制 / 输出资产 / 应用证据 / 失败模式与结论）
- 语言：简体中文（`ctexart` + `xelatex` + `unsrtnat`）；编译 0 错 0 警告

## 三、与现有英文综述的差异

近期已有若干英文 3R 综述（多数聚焦多视图几何或神经辐射场综合）。本综述在三点上独立成稿：

1. **中文学术语言**：服务中文阅读群体；术语对齐《计算机视觉》《自动化学报》等中文期刊近期 3R / SfM 类论文的用词习惯
2. **聚焦前馈范式**：明确把研究边界锁定在「不依赖测试时迭代优化即可输出几何」的方法族，把测试时优化方法（Test3R / TTT3R 等）作为辅助而非主线讨论
3. **失败模式系统化**：第 10 章按弱纹理 / 镜面玻璃 / 快速运动 / 长基线 / 尺度漂移 / 域外六类典型失败模式作系统分类，每类配可观察现象与现有缓解机制综述

## 四、证据边界标注

为避免「读到论文宣称即视为能力已证实」的偏差，第 9 章引入四层证据阶梯：

- 论文证据：仅论文文本与图表支持
- 官方代码证据：作者发布的代码可复现
- 跑通记录证据：第三方独立跑通并报告结果
- 应用验证证据：在下游任务（SLAM / 重建管线等）中被采用且报告效果

每个被引方法在第 9 章 `tab:application` 中按所达到的最高证据层级标注。综述本身的判断只对论文证据层负责；跑通记录与应用验证证据来源于外部观察，已标注来源。

## 五、路线说明

本综述按 arXiv 自存档（self-archiving）路线发布，不投递期刊或会议。理由：

- 综述的核心价值在于把握 2026 年中节点的方法分布，时效优先于评审周期
