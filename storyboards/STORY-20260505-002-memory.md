## Narrative Beats

ordered_steps:

```text
1. start state: a 48-frame walk through a scene begins. Frame 1 has
   landmarks the system should remember.
2. trigger: streaming continues. Frames 16-24 contain hard-to-reconcile
   regions; latent_drift_proxy rises.
3. action: Dream Memory budgets anchors — keeps the frame-1 landmarks,
   drops noisy intermediate writes (Permanence's suppress_static_write
   helps).
4. action: comparator (write-everything) writes every frame's state;
   memory grows without governance.
5. action: comparator (fixed-window) drops the frame-1 landmarks
   entirely once they pass the window.
6. visible result at frame 48: Dream Memory's reconstruction is still
   anchored to frame-1 landmarks; both comparators have drifted or
   noise-polluted their memory.
7. takeaway: governance ≠ more memory. Governance = the right memory.
```

surprise_hook:

```text
"Memory that knows what to drop survives a walk where memory that
keeps everything drowns."
```
